# Friend Lookup API

Friends data lookup API using Django REST Framework.

## Features
- GET friends data (id, name, dob, facebook, instagram)
- POST
- PUT/UPDATE
- DELETE

## Thanks:

Django REST Framework: https://www.django-rest-framework.org/
